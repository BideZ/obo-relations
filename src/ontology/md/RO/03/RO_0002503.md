
## Class : towards

 * *IRI* = http://purl.obolibrary.org/obo/RO_0002503
 * *Label* = towards
 * *Definition* = q towards e2 if and only if q is a relational quality such that q inheres-in some e, and e != e2 and q is dependent on e2
